Le streaming arrive sur l'API

Avec la v1.4, les réponses arrivent token par token via Server-Sent Events, sur l'API comme dans les SDKs Python et TypeScript. Le premier caractère de code s'affiche désormais en moins de 300 ms.

C'était la demande n°1 des développeurs qui intègrent HenergyqueAI dans leurs outils internes. Activez-le simplement avec "stream": true — le format des événements est décrit dans la documentation.

Pourquoi un modèle 7B spécialisé bat un généraliste géant

Un modèle compact entraîné exclusivement sur du code n'a pas à « penser » à la poésie ou à la cuisine : chaque paramètre sert la génération, le refactoring et le debug. Résultat : des réponses plus rapides, moins chères, et exécutables en local.

Dans cet article, on détaille nos choix de jeux de données, notre pipeline de fine-tuning et les benchmarks internes qui nous ont convaincus de parier sur la spécialisation plutôt que sur la taille.
